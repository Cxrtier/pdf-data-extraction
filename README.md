# PDF Data Extraction Pipeline

Automated pipeline to extract and structure text data from large PDF 
catalogues using Python.

## What it does
- Extracts text content page by page from PDF files
- Handles missing content, empty pages, and file validation
- Processes large document sets containing millions of product entries

## Built for
Data Science internship at MENO Group — processing hundreds of product 
catalogues for a DIY hardware store.

## Tech used
- Python
- PyPDF2

## How to run
pip install PyPDF2
python extract.py
